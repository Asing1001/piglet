# Piglet

- Theme: [ScrewFast](https://github.com/mearashadowfax/ScrewFast/)

## Prerequisite

1. [install nvm](https://github.com/nvm-sh/nvm?tab=readme-ov-file#installing-and-updating)
1. vercel cli, `npm i -g vercel`

## Getting Start

## First time Setup

1. Clone the repository
1. `nvm use`
1. `npm i`
1. `vercel link`
1. `vercel env pull`

## Development

`npm run dev`
